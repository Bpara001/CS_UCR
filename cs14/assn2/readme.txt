This word ladder works ONLY with the specifications noted in the assn:
	-words most be EXACTELY 5 letters long,
	-use the dictionary, "words5.dict", as given in the assn specs
	
To run program, type in the arguments,
	<executable> <dictionary> <5_letter_word> <5_letter_word
	ex: $a.out words5.dict style crazy

Note, the program uses static arrays instead of lists
	-it was not stated to make the most efficent program
	-or to use the most efficent data structure

For the given word ladder examples, such as
style->crazy, brave->heart, stack->steak, black->black
it takes roughly 5-15 seconds to run so be patient
